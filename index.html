FAQ

Pertanyaan yang
Sering Ditanyakan

Apakah ada biaya setup atau biaya tersembunyi?
Tidak ada biaya setup dan tidak ada biaya tersembunyi sama sekali. Kamu hanya membayar biaya berlangganan bulanan atau tahunan sesuai paket yang dipilih. Tidak ada komisi per transaksi, tidak ada biaya aktivasi, dan tidak ada kontrak jangka panjang.
Bagaimana cara kerja QR Menu Digital?
Setiap meja mendapatkan QR code unik. Pelanggan cukup scan QR code menggunakan kamera smartphone (tidak perlu download aplikasi), lalu menu digital akan muncul di browser mereka. Pelanggan bisa memilih menu, menambahkan ke keranjang, dan melakukan pembayaran langsung dari smartphone mereka.
Apakah bisa digunakan tanpa koneksi internet?
Tantri memiliki fitur offline mode untuk kasir. Jika koneksi internet terputus, kamu masih bisa memproses transaksi dan data akan otomatis tersinkronisasi saat koneksi kembali. Namun untuk fitur QR Menu, koneksi internet diperlukan agar pelanggan bisa mengakses menu digital.
Berapa lama proses setup dan onboarding?
Proses setup sangat cepat! Rata-rata bisnis bisa mulai beroperasi dalam 1-2 jam setelah daftar. Tim onboarding kami akan memandu kamu melalui proses setup menu, konfigurasi kasir, dan cara mencetak QR code. Kami juga menyediakan video tutorial dan dokumentasi lengkap.
Metode pembayaran apa saja yang didukung?
Tantri mendukung berbagai metode pembayaran: Tunai, QRIS (semua e-wallet), GoPay, OVO, Dana, LinkAja, ShopeePay, Transfer Bank (BCA, BNI, BRI, Mandiri), Kartu Debit/Kredit (Visa/Mastercard), dan Paylater. Semua terpusat di satu dashboard pelaporan.
Apakah data saya aman di Tantri?
Keamanan data adalah prioritas kami. Semua data dienkripsi menggunakan SSL/TLS 256-bit dan disimpan di server berstandar ISO 27001. Kami melakukan backup otomatis setiap hari dan memiliki uptime 99.9%. Data kamu tidak akan pernah dijual atau dibagikan ke pihak ketiga.
Bagaimana jika saya ingin berhenti berlangganan?
Kamu bisa berhenti berlangganan kapan saja tanpa denda atau penalti. Untuk paket bulanan, kamu masih bisa menggunakan layanan hingga akhir periode berlangganan. Kami juga memberikan garansi uang kembali 30 hari jika kamu tidak puas dengan layanan kami.
